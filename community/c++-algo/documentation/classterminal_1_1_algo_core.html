◆ start()

void terminal::AlgoCore::start ( )
virtual

Starts the main loop of the program.

This will continue until it gets the end of state signal from the engine. If there is some error, it will never exit since it will continue to wait until it recieves input from the engine. If this happens it must be killed manually.
